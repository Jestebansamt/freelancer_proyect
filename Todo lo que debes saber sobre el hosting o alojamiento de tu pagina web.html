Alojamiento web o hosting


Publica tu sitio web

Ten encuenta varios aspectos a la hora de escoger un nombre de dominio

    Nombre

  • El nombre para tu pagina web debe ser preciso y explicativo, para tener un mejor posicionamiento en los buscadores que a su vez te dara mas visitas las cuales se traducen mas ganancias u oportunidades de monetizar tu trabajo (publicidad)
  • Existen nombres de dominios no disponibles como por ejemplo Google.com o YouTube.com, se creativo busca nombres llamativos .

    • Extensión

  • Entre mas reconocida sea la extensión (.com .net ) el precio de adquisicion por el dominio va a aumentar.

    • Tiempo

  • El dominio no se vende por tiempo ilimitado lo mas normal es que consigas planes anuales o mensuales. Asi que ten cuidado y memoriza la fecha en la que debes actualizar tu dominio para que nadie mas se te adelante

Para tener en cuenta en tu alojamiento o servidor web

    Servidor

  • Como servidor web existen Servicios para almacenar informacion Web ya sean, paginas, correos, etc.

    • Alojamiento

  • Como alojamiento para tu sitio web existen alternativas gratuitas las cuales son recomndables para practicar o probar diseños. Por otra parte, si se quiere un alojamiento web para un sitio profecional lo mas aconsejable es pagar por uno adecuado para evitar publicidad fastidiosa y tener mejores herramientas.

Opciones para servidores o alojamientos web pagos

  • Existen dos alternativas que se pueden presentar en cada uno de los servidores que seran mencionados en esta seccion. La primera es el hecho de que la infromacion puede ser almacenada en una instalacion fisica como por ejemplo un monitor. La segunda es en la nuve en donde los servidores son complretamente virtuales, son mas seguros y mas flexibles gracias a que tiene mas prestaciones

    • Compartido:

  • Es el mas comun debido a su relacion calidad precio, adicionalmente no tienes que centrarte en la manutencion del servidor. Sin embargo compartes los recursos con otros alojamientos indepentientes al tuyos. Normalmente se utiliza para sitios web que tienen un trafico pequeño. Wikipedia

    • Servidor virtual privado (VSP):

  • Es una opcion mas segura y estable que el servidor compartido en otras palabras se tiene mas control del sitio web a un precio mayor, normalmente se usa para sitios web que tienen un trafico medio. Para mas info Hostinger Tutoriales

    • Servidor dedicado:

  • Mejores prestaciones en el alojamiento y control completo del servidor, la diferencia fundamental que tiene con el VSP es que la informacion se guardan en espacion fisicos, en un ordenador que puede ser comprado o alquilado, es decir necesitas de los complementos materiales, ademas de un gran conocimiento en la manutencion de los servidores. Para mas info ¿VPS o Dedicado?




Criterios para comprar HOST

Calidad

  • Que tanta seguridad te da el sitio web en relacion a que cumpla con todo lo que te prometieron antes de la contratacion
  • Debera resolver efectivamente los problemas que tengamos
  • Tiene que tener buena disponibilidad(que no este apagado frecuentemente)

Indicadores para medir la calidad del servicio de Alojamiento o Hosting

  • Calidad web de la empresa, el aspecto que tenga la pagina web hablara mucho sobre la empresa
  • Opiniones de los clientes
  • Tiempo operando, la experiencia es fundamental para estos escenarios, aunque pueda sonar prejuicioso tengamos en cuenta que a mayor tiempo operando la experiencia sera mayor, adicionalmente si ha podido prosperar en el mercado x tiempo es por que tiene algo bueno que ofrecer a sus clientes
  • Soporte al cliente, el tiempo de respuesta que debe tener la empresa contratada debe ser minimo frente a cualquier problematica que se pueda llegar a presentar
  • Buscar comparativas recientes que nos ayuden a tomar una mejor decision

Sistema Operativo

  • Es importante tener en cuenta que segun el sistema operativo en el que estes trabajando vas a necesitar un servicio diferente. En el caso de Linux el alojamiento web tiende a ser mas economico que el host en Windows.

Almacenamiento

    ¿Que incluye?

  • Puede inclur bases de datos o plantillas que abarquen todo el espacio gratuito o pago disponible obligandonos a pagar mas, los que tengan esta caracteristica no tienen mucha seriedad como empresa

    • ¿Que almacenamiento necesito?

  • Se realiza un calculo de aproximacion, el cual se aumentara un 20%
  • El calculo de aproximacion se saca multiplicando el numero de paginas que tenga el sitio web por el Tamaño medio de la pagina
  • Para sacar el tamaño medio de la pagina es necesario hacer una suma de todos los archivos que tenga la pagina web, CSS, HTML, JavaScvript e imagenes

Transferencia

  • La unidad por la cual se mide son gigabytes/mes la cual calcula la capacidad del volumen de transferencia que tiene el host
  • El volumen de transeferencia es dificil de calcular ya que mide el numero de visitas que tendra la pagina web
  • Aunque algunas empresas usan el slogan de "Transferencia ilimitada" no es correcto debido a que se necesita un lugar fisico para almacenar dichas trnaseferencias

    • ¿Que volumen de transferencia necesito? y ¿Como calcularlo? aprox

  • Multiplicar numero de visitantes X paginas X usuario
  • El resultado multiplicarlo X el tamaño medio de la pagina
  • El resultado multiplicarlo X 30 (dias del mes)

    Velocidad de transeferencia

  • Ancho de banda
  • No existe un ancho de banda ilimitado
  • Unidad: Kbps - Mbps
  • Entre mas ancho de banda el tiempo de espera por parte del usuario sera menor
  • El ancho de banda es compartido por el numero de visitantes que haya al mismo tiempo en la pagina

Dominios

    Subdominios

  • Divisores del dominio, por ejemplo carros/azules.com, en este caso /azules seria un subdominio del dominio carros.com

    • Dominios

  • Nombre para el sitio web
  • Se pueden usar varios dominios para una sola pagina web como estrategia de marketing

Otros
