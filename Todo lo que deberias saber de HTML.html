    HTML5

  • HTML5 es la quinta versión de un lenguaje de programación destinado al desarrollo de páginas web, sus siglas en ingles son el acrónimo de Hyper Text Markup Language, algo así como Lenguaje de marcado de hipertexto. Pero ¿De dónde proviene este nombre? El Hipertexto son los enlaces que conectan las diferentes páginas web, por ejemplo, “YouTube”. Los enlaces dentro de los sitios web conectan y hacen mucho más accesible la web. El marcado de texto hace referencia a la manera en la que se redacta el código ya que los elementos en HTML se escriben entre dos <> como imágenes <"img"> artículos <"article"> o incluso el tamaño de la letra <"h1"> <"h6">, estas etiquetas son las que se utilizan para escribir código en HTML y se diferencian unas de otras por medio de los símbolos mayor que “>” y menor que “<”.


    • CSS3

  • Al igual que HTML5 el numero hace referencia a la versión actual del lenguaje. CSS es el acrónimo en inglés de Cascading Style Sheets. Este lenguaje ha estado muy ligado desde su creación a HTML por lo que se complementan bien al momento de desarrollar sitios web.CSS es un lenguaje puramente visual, desde ahí es donde se especifica desde el ancho de las letras hasta las dimensiones que va a tener la página web.
