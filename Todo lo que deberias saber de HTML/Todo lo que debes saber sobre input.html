Nombre Escrito en codigo Uso Visualizacion Atributos Visualizacion con atributos
Name Es un atributo. Para mostrar texto y Nombre del campo para envio. Normalmente solo el programador ve el nombre del cuadro de texto.
Type Es un atributo. Para especificar el tipo de cuadro que se visualizara. Normalmente solo el programador ve el nombre del cuadro de texto.
Cuadros de texto input type="text" name="Elquesedesee" Para mostrar cuadros de texto... Maxlength ( max de letras). Size (Maximo de espacio que se puede ocupar).
Checkbox input type="Checkbox" name="Checkbox" Para genera listas con check checked (marca la opcion por defecto). Value (informacion que debera ser enviada en caso de que se marque la casilla)
Casilla de opcion radio input type="radio" name="radioo" Generar listas con check redondas Los mismos que checkbox
Campos de texto enmascarado input type="password" name="paswword" Muestra campos de texto a modo de clave
Botones input type="button" name="boton" Para generar botones con texto que pueden redirigir a otras paginas. Value (texto dentro del boton). Entre muchisimas otras
