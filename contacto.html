Contacto


    ¿Quién soy?

  • Mi nombre es Juan Esteban Santacruz y trabajo como FreeLancer o trabajador independiente. Este es mi primer proyecto como programador, y mi objetivo con él es diseñar y programar páginas web 100% personalizables. Me especializo en HTML5 Y CSS3, lenguajes fundamentales para el desarrollo de páginas web; estos son los lenguajes que utilicé para la construcción de este sitio web.


    • ¿Cómo es el proceso para construir una página web?

  • Como programador web me encargo de hacer una o varias reuniones con el cliente para definir el diseño y la estructura del sitio web. Durante la escritura del código, desarrollar un sitio web con diseño adaptativo, de tal manera que no haya ningún inconveniente si se quiere visualizar en un computador, en un celular o en una tableta, lo que permitirá que la página web tenga una mayor accesibilidad. Y finalmente publicar el sitio web y la manutención del mismo.


    • ¿Qué necesitas saber?

  • Es importante que tengas claro lo que quieres, como quieres que sea tu página web, puedes buscar inspiración de muchos sitios web o yo te mostrare muchas plantillas que podrás usar como inspiración para la construcción de tu página. Aunque la decisión del diseño es plenamente del cliente yo te ayudare a tener un diseño estético, organizado y claro, de esta manera la información dentro de la página web será lo más agradable y accesible para los usuarios.
