Contacto


    ¿Quién soy?

  • Mi nombre es Juan Esteban Santacruz y trabajo como FreeLancer o trabajador independiente. Este es mi primer proyecto como programador, y mi objetivo con él es diseñar y programar páginas web 100% personalizables. Me especializo en Html5 Y Css3, lenguajes de programación fundamentales para el desarrollo de páginas web; estos son los lenguajes que utilicé para la construcción de este sitio web.


    • ¿Qué hago?

  • En pocas palabras, me encargo del diseño de la página web, la escritura de código para implementar el diseño y la publicación de la página web como paso final. Yo, al igual que muchos programadores, me encargo en hacer un diseño adaptativo de las páginas, de tal manera que no haya ningún inconveniente si se quiere visualizar en un computador, en un celular o en una tableta, lo que permitirá que la página web tenga una mayor accesibilidad.


    • ¿Cómo trabajo?

  • La principal herramienta para que una página web sea exitosa no es únicamente la cantidad de código que requiera o el presupuesto de inversión que se tenga. El éxito de una página web radica en la comunicación entre el programador y el cliente: con una comunicación eficiente, el resultado del proyecto será mucho más satisfactorio.
